About Me

After earning my B.S. in Geographic Information Systems in 2013, I began to work as a GIS Technician for the County of San Diego and Royale Energy Inc. During this time I noticed several opportunities for software to increase efficiency and productivity in the workplace, so I began to learn the basics of programming and software engineering in my free time. Eventually I wrote a few python scripts to improve the efficiency of my work flow. The more I learned about programming, the more I enjoyed it and this passion continued to grow until I decided to pursue a M.S. in Computer Science at San Diego State University, which I completed in 2019. I have a proven record, both academically and professionally, of exhibiting a willingness and ability to learn quickly, solve problems and complete tasks.

I currently work at General Atomics, Aeronautical Systems as a software developer where I work on software which runs in our ground control stations and communicates with unmanned aeirial vehicles.

When I'm not working I enjoy cooking, reading, attempting (and often failing) to tire out my two year old dog, playing video games and spending time with my family.

Projects

Aztec Study Place

[Android (Java)]

I made this application while I was a graduate student at SDSU. It allows users to login with their university account, then downloads their class schedule and finds blocks of time the physical classrooms thier lectures are in are not in use so they can go study in them. Additionally, there is a feature which allows users to find classrooms which are currently available to study in near them. The motivation behind this was the idea of Context-dependent Memory which is "the improved recall of specific episodes or information when the context present at encoding and retreival are the same."

Vacation Tracker

[Java, Swing]

This is a Java desktop app which gets flight information from a user via a Swing GUI. I then (would) use the SkyScanner API to keep track of flights entered and look for price drops to alert the user via the Twilio API. I decided to make this because I like saving money and I wanted to learn the basics of Java GUI design and how to add basic persistence to my apps. Unfortunately the SkyScanner API is no longer free, so this application does not work as intended.
